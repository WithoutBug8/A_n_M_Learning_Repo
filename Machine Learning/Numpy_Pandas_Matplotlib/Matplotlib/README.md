# 这是Matplotlib的学习笔记

### 简介
Matplotlib 是 Python 的一个绘图库，它提供了一整套功能完备的绘图工具，用于创建静态、动画和交互式可视化。
Matplotlib 的设计哲学是“一个接口，多个后端”，这意味着你可以使用相同的代码在不同的输出格式（如 PDF、PNG、SVG 等）和 GUI 后端（如 Tk、Qt、GTK 等）之间进行切换。

| 工具 | 说明 | 优点 | 缺点 |
| :--- | :--- | :--- | :--- |
| matplotlib | Python 最基础可视化库 | 灵活强大、定制性强 | 代码多、风格基础 |
| seaborn | 基于 matplotlib 的高级接口 | 风格美观、统计图方便 | 对简单图略繁琐 |
| pandas plot | 快速图表，调用 .plot() | 快捷、适合 EDA | 图表样式较少 |
